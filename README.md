📚 AI Exam Notes Generator

An AI-powered SaaS web application that generates smart, concise, and structured exam notes from user input using modern AI APIs. Built with the MERN Stack for students who want quick revision material and efficient learning.

🚀 Features
✨ AI-generated exam notes
📄 Upload topic or enter prompt
🧠 Smart summarization using AI
🔐 User Authentication & Authorization
💾 Save generated notes
📱 Responsive modern UI
⚡ Fast and optimized performance

⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/your-username/ai-exam-notes-generator.git
2️⃣ Navigate to project folder
cd ai-exam-notes-generator
3️⃣ Install dependencies
Frontend
cd client
npm install
Backend
cd server
npm install
🔑 Environment Variables

Create a .env file inside the server folder:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_api_key
▶️ Run the Project
Start Backend
cd server
npm run dev
Start Frontend
cd client
npm start
